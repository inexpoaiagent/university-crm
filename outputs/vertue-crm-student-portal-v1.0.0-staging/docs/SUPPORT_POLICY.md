# Support Policy

## Support Channel
- CodeCanyon comments and direct support channel listed on item page.

## Response Time Target
- Within 24 business hours (Mon-Fri).

## What Support Includes
- Installation guidance
- Environment configuration guidance
- Reproducible bug fixes for included core features
- Clarification of documented functionality

## What Support Does Not Include
- Custom feature development
- Design/content customization work
- Third-party plugin integration not included in package
- Server administration unrelated to application code

## Requirements for Bug Reports
Please provide:
- Product version
- PHP version
- Database type/version
- Exact steps to reproduce
- Full error message and stack trace
- Relevant screenshots

## Update Policy
- Maintenance and bug-fix updates are released as needed.
- Minor feature improvements may be included in periodic updates.
- Breaking changes, if any, are documented in changelog.
