# Support Policy

## Included
- Installation guidance
- Setup and configuration guidance
- Reproducible bug fixes for bundled features
- Clarification for documented modules

## Not Included
- Custom feature development
- Custom UI redesign outside bundled package
- Third-party paid integration implementation
- Full server administration

## Response Time
- Target: within 24 business hours (Mon-Fri)

## To Get Fast Support
Send:
- Product version
- PHP and MySQL versions
- Exact error message
- Reproducible steps
- Relevant screenshots
